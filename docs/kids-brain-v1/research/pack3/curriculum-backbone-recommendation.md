# Curriculum Backbone Recommendation
## Mentium Kids — Ages 6–8

---

## 1. Textbook/Course Comparison

### Evaluation Criteria
- Voice-first compatibility (can content be delivered aurally without visual page dependency)
- Vocabulary pacing (words per unit, recycling rate)
- CEFR progression clarity
- Song/chant/rhythm content (critical for young learners)
- Real-world topic relevance for 6–8
- Grammar spiral structure
- Suitability for non-native-English-context learners
- TPR integration potential
- AI lesson scalability

---

### Candidate 1: Cambridge Super Minds (Starter / Level 1)
**Publisher:** Cambridge University Press  
**Target age:** 6–8 (Starter), 7–9 (Level 1)  
**CEFR:** Pre-A1 (Starter) → A1 (Level 1)

**Strengths:**
- Explicit "Thinking Skills" strand integrates non-linguistic reasoning — reduces cognitive overload
- Dedicated "Values" strand builds emotional/social context relevant to child world
- Strong phonics sequence (especially Starter)
- Vocabulary is tightly controlled: ~6–8 new words per unit, very high recycling ratio
- Grammar introduced through meaning-first chunks ("It's a ___", "I like ___", "This is my ___")
- Songs and chants professionally produced, age-calibrated
- Cambridge alignment means CEFR progression is defensible to parents
- Digital component well-developed (helpful for understanding what works in edtech)
- Interleaved review units every 3 units

**Weaknesses:**
- Visual-heavy textbook UX (pages, stickers, coloring) — irrelevant/inaccessible in voice-first
- Some writing activities — irrelevant for Mentium
- Stories slightly slow-paced for 6-year-olds in AI context
- Grammar labels sometimes appear explicitly — not recommended for AI teacher

**AI adaptability score:** ★★★★☆

### Candidate 2: Cambridge Kid's Box (Updated 2nd Edition, Starter / Level 1)

**Target age:** 6–8

## 2. Recommended Choice

### ✅ PRIMARY: Cambridge Kid's Box Updated 2nd Edition (Starter + Level 1)

**Secondary reference:** Cambridge Super Minds (Starter) for phonics sequencing and thinking skills tasks

**Rationale:**

Kid's Box wins for Mentium Kids because:

1. **Vocabulary control is best-in-class.** ~6 new words per unit, appearing in stories, songs, games, and review activities. This recycling rhythm is exactly what an AI short-cycle lesson format needs.

2. **Recurring characters (Marie, Stella, Simon, Maskman) create emotional continuity** without visual dependency — they can be voice-acted or referenced in audio. This gives the AI teacher a "story world" to anchor lessons.

3. **Grammar is introduced as communicative chunks, not rules.** "I've got a ___." "What's in the ___?" These are speech acts, not grammar tables.

4. **Song/chant corpus is large and pedagogically embedded** — not decorative. Chants encode target vocabulary in rhythm, which is a proven retention mechanism for this age. [A]

5. **Cambridge Young Learners exam alignment** gives parents a visible external benchmark. This is a retention/trust mechanism.

6. **Phonics strand is explicit** — important because ages 6–8 are in active literacy development. Even in voice-first, letter-sound awareness supports pronunciation learning.

7. **Topic sequence maps perfectly to 6–8 cognitive and emotional world:** family, animals, toys, food, school, body, clothes, transport, weather. These are experientially real for the child.

8. **Review every 3 units** matches short-cycle AI lesson rhythm — built-in recycling without curriculum redesign.

## 3. Exact Age and Level Recommendation

| Child Age | Recommended Starting Level | Notes |
|-----------|---------------------------|-------|
| 6 (true beginner) | Kid's Box Starter | Move slowly: 2–3 lessons per unit |
| 6–7 (some exposure) | Kid's Box Starter → Level 1 | Unit completion signals readiness |
| 7–8 (some exposure) | Kid's Box Level 1 | Begin at Unit 1, progress by mastery |
| 8 (strong) | Kid's Box Level 1 → Level 2 | Bridge into Level 2 after 8–10 units |

**Placement signal:** Not a formal test. First lesson diagnostic: can the child repeat 3 words after hearing them once? Can they point (or say) the right answer from a 2-item forced choice? That determines Starter vs Level 1 entry.

## 4. Why It Fits Ages 6–8

[A] **Working memory:** Ages 6–8 can hold approximately 4 ± 1 items in working memory. Kid's Box Starter introduces 4–6 words per unit. This matches working memory constraints exactly.

[A] **Attention span:** Sustained voluntary attention at age 6 averages 10–15 minutes. Kid's Box unit activities average 5–8 minutes. This maps to Mentium's micro-cycle design.

[B] **Whole-language vs phonics:** Ages 6–7 benefit most from phonics-embedded reading, but oral language first. Kid's Box delivers vocabulary orally (song, story, game) before any written encounter.

[A] **Emotional scaffolding:** Ages 6–8 need identity-safe learning. Kid's Box characters are relatable children, not authority figures. This reduces performance anxiety.

[B] **Meaning-before-form:** Kid's Box consistently shows language in context (story, song, play) before any focus on form. This aligns with comprehensible input research (Krashen) and communicative language teaching principles.

## 5. How to Use It Legally and Safely

**Critical:** Mentium Kids does NOT reproduce textbook content. Do NOT:
- Copy song lyrics
- Reproduce story text
- Use character art
- Digitize illustrations
- Copy exercise instructions verbatim

**What Mentium extracts (legally safe):**
- The **topic sequence** (Animals → Colours → Family → Food...) — this is curriculum structure, not copyrighted expression
- The **vocabulary lists per unit** — word lists are not copyrightable
- The **sentence pattern types** (e.g., "I've got a ___" is a language pattern, not creative expression)
- The **vocabulary recycling logic** (when words reappear across units)
- The **CEFR level mapping** — this is a public framework

**Legal safe harbor:** Mentium creates entirely original:
- Story scripts using the same topics
- Original songs/chants using the same vocabulary
- Original games/activities targeting the same language
- Original character voices and worlds

This is parallel creation, not reproduction. This is how every licensed ESL platform (Lingokids, Novakid, etc.) operates.

- Any verbatim text, dialogue, or story from the textbook
- Any illustration description that would recreate visual IP
- Any exercise format that is distinctively the publisher's creative expression
- Any audio script from the official recordings
- Character names, storylines, or backstories from Kid's Box

## 8. How It Maps to AI Teacher Runtime

```
CURRICULUM BACKBONE (Kid's Box structure)
         ↓
UNIT DEFINITION (vocabulary list, sentence patterns, topic, CEFR level)
         ↓
LESSON PLAN (activity sequence from lesson-format-spec.md)
         ↓
ACTIVITY EXECUTION (primitives from activity-library.yaml)
         ↓
TEACHER BEHAVIOR (rules from kids-brain-rules.yaml)
         ↓
CHILD RESPONSE → ASSESSMENT SIGNAL
         ↓
NEXT ACTIVITY SELECTION (or recovery path)
```

The curriculum backbone feeds the **content layer** only. The AI teacher runtime is built on top of it, independently. The backbone defines WHAT to teach. The lesson format defines HOW to teach it.
