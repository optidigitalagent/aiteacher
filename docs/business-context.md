## ЧТО ЭТО ЗА БИЗНЕС

**Продукт:** Голосовой AI-учитель английского для школьников 12–17 лет.
Заменяет репетитора. Работает 24/7. Стоит в 10–30x дешевле живого учителя.
 
**Рынок:** AI Tutoring Services — $1.63 млрд в 2024, растёт до $8 млрд к 2030.
CAGR 30.5%. Это один из самых быстрорастущих сегментов EdTech.

**Конкурентное преимущество:**
- Duolingo / Babbel — геймификация, нет реальной педагогики, нет голоса
- ChatGPT — нет структуры урока, нет учебника, нет Student Model
- Репетитор — $30–100/час, не масштабируется, нет 24/7
- Наш продукт — структурированный урок + голос + учебник + адаптация + $15–30/мес

## КТО ПЛАТИТ (Customer Segments)

### Сегмент 1: Родители школьников (B2C — основной MVP)
- Платят за ребёнка 12–17 лет
- Мотивация: заменить/дополнить репетитора, подготовка к ЕГЭ/ОГЭ/IELTS
- Платёжеспособность: $15–40/мес это меньше одного урока с репетитором
- География MVP: Украина, Россия, Казахстан, Беларусь (рус. язык интерфейса)
- Решение о покупке: мама/папа, триггер — оценки в школе

### Сегмент 2: Языковые школы (B2B — после MVP)
- Покупают как дополнение к офлайн урокам: "домашнее задание с AI"
- Платят за лицензию: $200–500/мес за группу учеников
- Ценность: снижают нагрузку на учителей, дают ученикам практику вне класса

### Сегмент 3: Школы и гимназии (B2B Enterprise — долгосрочно)
- Школа покупает доступ для всех учеников класса/параллели
- Контракт: $5,000–50,000/год
- Требуют: аналитику прогресса, интеграцию с учебником, отчёты для учителей

## МОДЕЛЬ МОНЕТИЗАЦИИ

### MVP: Freemium + Subscription

```
FREE (пробный)
├── 2 бесплатных урока
├── Только текстовый режим (без голоса)
├── Один юнит из учебника
└── Цель: дать почувствовать ценность → конвертировать

BASIC — $15/мес
├── 8 уроков в месяц (2/неделя)
├── Голос + текст
├── Один учебник (Focus B1)
└── Базовый прогресс-трекер

PRO — $29/мес
├── Неограниченные уроки
├── Все уровни (A2 → B2)
├── Все учебники
├── Расширенная аналитика для родителей
├── Spaced repetition словарь
└── Приоритетная обработка (меньше задержка)

SCHOOL — $199/мес (до 30 учеников)
├── Все PRO функции
├── Дашборд учителя: прогресс всего класса
├── Экспорт отчётов
└── Кастомные юниты под программу школы
```

### Юнит-экономика (цели)
```
CAC (стоимость привлечения): $8–15 (таргет в соц. сетях на родителей)
LTV Basic:  $15 × 6 мес = $90
LTV Pro:    $29 × 12 мес = $348
LTV/CAC:    минимум 6:1 — цель 10:1
Churn:      цель < 8%/мес (ключевой метрик, зависит от качества урока)
```

```typescript

// teacher видит прогресс всех своих студентов

```

### 3. Аналитика для родителей → store всё в lesson_events
```
Родитель хочет видеть:
- Сколько уроков прошёл ребёнок на этой неделе
- Какие грамматические темы изучены
- Процент правильных ответов
- Слова которые он учит
→ Всё это берётся из lesson_events + exercises таблиц
→ НЕ НУЖНО отдельное хранение — просто правильные запросы
```

### 4. Freemium конверсия → upsell момент в уроке
```
После 2го бесплатного урока → AI говорит:
"You've completed 2 lessons. You're making great progress!
To continue, your parent can unlock unlimited lessons."
→ Показать paywall с ценами
→ Это решение в LessonOrchestrator: проверять лимит перед запуском урока
```

## КЛЮЧЕВЫЕ МЕТРИКИ ДЛЯ ВЫЖИВАНИЯ СТАРТАПА

Claude Code должен знать это чтобы понимать ПРИОРИТЕТЫ разработки:

```
1. COMPLETION RATE (важнейший)
   % уроков, которые доходят до фазы WRAP_UP
   Цель: > 75%
   Если < 50% → продукт не работает, нужен срочный фикс
   Хранить: lessons.phase_reached, lessons.status

2. WEEKLY ACTIVE STUDENTS
   Сколько студентов провели хотя бы 1 урок за неделю
   Цель: > 60% от активных подписчиков
   Зависит от: качества урока, напоминаний, привычки

3. LESSON QUALITY SCORE
   Средний score урока (0.0–1.0 из exercises таблицы)
   Цель: > 0.65
   Если падает → AI промпт нужно улучшить

4. VOICE LATENCY (технический, но бизнес-критичный)
   Время от конца речи ученика до начала ответа AI
   Цель: < 2.5 сек
   Если > 4 сек → ученики бросают урок
```

## КОНКУРЕНТЫ — КРАТКИЙ АНАЛИЗ

| Конкурент | Что делает | Слабость | Наше отличие |
|---|---|---|---|
| Duolingo Max | Геймификация + GPT-4 звонки | Нет структуры урока, нет учебника | Полный урок по школьной программе |
| Preply | Живые репетиторы | Дорого ($15–60/урок), нет 24/7 | AI + 10x дешевле + всегда доступен |
| Khanmigo | AI объяснения Khan Academy | Только текст, нет голоса, нет School программы | Голос + российский рынок + Focus |
| Busuu | Курсы + AI feedback | Нет живого диалога, нет адаптации под урок | Диалог в реальном времени |
| ChatGPT | Всё что угодно | Нет педагогики, нет Student Model, нет учебника | Структура + память + адаптация |

**Главный инсайт:** никто не делает полноценную замену репетитора
с голосом + конкретным учебником + педагогической структурой.
Именно это наш продукт.

## РИСКИ (знать чтобы не строить лишнее)

**Технический риск #1:** Задержка голоса > 3 сек → убивает опыт
→ Решение: streaming TTS + Deepgram Nova-2 (самые быстрые)

**Бизнес-риск #1:** Churn > 15%/мес → не окупается CAC
→ Решение: качество урока + напоминания + прогресс-трекер видимый родителям

**Правовой риск:** Дети < 13 лет — COPPA/GDPR-K
→ Решение: на MVP не принимать детей младше 12, consent от родителей

**Конкурентный риск:** Duolingo/Google копируют
→ Решение: глубокая интеграция с конкретными учебниками (это сложно скопировать быстро)
