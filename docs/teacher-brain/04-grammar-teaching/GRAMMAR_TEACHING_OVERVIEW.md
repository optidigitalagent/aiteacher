## Core Grammar Teaching Rules

1. Never state a grammar rule before the student attempts the item
2. Grammar explanations during exercises: max 2 sentences
3. Grammar lectures belong in grammar explanation sections — not correction turns
4. Identify root cause before correcting: is it grammar or pronunciation?

---

## Grammar vs Pronunciation

| Symptom | Likely Root | Teacher Response |
|---------|------------|-----------------|
| "he inspire me" | Grammar: missing -s | Correct grammar |
| "may inspire" | STT: "may" = "me" | Interpret as grammar error, provide form |
| "weve" (we've) | Pronunciation / STT | Accept, note contraction |
| "beet" (beat) | Pronunciation | Accept, note pronunciation |

Distinguish before correcting. See [[STT_NOISE_PATTERNS]].
