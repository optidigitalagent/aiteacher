# Skill: Update Master Prompt

## Цель
Безопасно изменить AI-промпт учителя без сломать педагогическую логику.
Файл: `docs/master-prompt.md`

## ПЕРЕД любым изменением

### 1. Прочитай полный файл
```
Read docs/master-prompt.md
```
Определи ФАЗУ которую затрагивает изменение:
- DIAGNOSTIC — первичный диалог, выяснение уровня
- CONTEXT_INPUT — сбор контекста (тема, интересы)
- RULE_DISCOVERY — студент сам находит правило
- EXERCISES — выполнение заданий
- VOCABULARY — работа со словарём
- DEEP_THINKING — анализ ошибок
- WRAP_UP — итог урока

### 2. Проверь Forbidden Changes
- [ ] Правило "NEVER say Wrong" осталось нетронутым
- [ ] Сократов метод (AI не называет правило первым) сохранён
- [ ] JSON output format не изменён
- [ ] `max_tokens` не превышает 400

### 3. Проверь токены промпта
```
Total system prompt:     MAX 4000 tokens
RAG context:             MAX 800 tokens (top 3 Pinecone chunks)
Conversation history:    last 8 exchanges ONLY
Student profile:         MAX 200 tokens (краткое резюме, не raw JSON)
```

### 4. Тест перед коммитом (3 сценария обязательно)
Протестируй с:
- **Правильный ответ:** студент ответил верно → AI хвалит + двигается дальше
- **Неверный ответ:** студент ошибся → AI не говорит "Wrong", использует Сократов метод
- **Запутанный студент:** "I don't understand" → AI объясняет иначе, не повторяет то же

## Шаги изменения

1. Read файл целиком
2. Определи фазу
3. Внеси минимальное изменение (не рефакторить всё)
4. Проверь Forbidden Changes выше
5. Посчитай токены (можно примерно: 1 token ≈ 4 символа)
6. Запиши что изменил и почему в Learnings

## Формат результата
Изменение в `docs/master-prompt.md` + запись в Learnings.
