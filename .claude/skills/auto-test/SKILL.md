# Skill: Auto Test → Fix → Verify

## Цель
Запустить полный автономный цикл:
тест → анализ → fix-промпт → фикс → верификация → Learnings.

## Когда использовать
- После любых изменений в backend TypeScript
- Перед деплоем
- По расписанию (daily health check)

## Шаги

### 1. Запусти auto-qa-loop агента
```
запусти auto-qa-loop
```
Агент сам пройдёт весь цикл.

### 2. Если агент вернул BLOCKED
Значит нашёл проблему которую нельзя фиксить автономно.
Читай его отчёт — там файл и рекомендация.

### 3. Если агент вернул ALL PASS
Можно деплоить. Запусти env-checker перед Railway:
```
запусти env-checker
```

### 4. Если деплой с миграциями
Сначала:
```
запусти migration-guard
```

## Полная цепочка перед деплоем
```
auto-qa-loop → (если PASS) → env-checker → (если READY) → deploy-railway
```

---

## Learnings

### Пример
**Что:** auto-qa-loop поймал отсутствие try/catch в новом обработчике
**Авто-фикс:** добавил try/catch за 1 итерацию
**Урок:** всегда запускай перед деплоем — экономит 20 минут ручного дебага
