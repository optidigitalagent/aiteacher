Follow `AGENTS.md` first. Use this skill before every CODE task in a product
goal.

## Output

```text
DEVELOPER REMINDER
Current user outcome:
Acceptance criterion served:
Implementation task:
Affected surfaces:
Protected surfaces:
Evidence required:
Stop if:
```

If the task does not map to an acceptance criterion, stop and run
goal-intake-orchestrator.
